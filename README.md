# 연합 학술제 기획 및 총괄 — GNN 기반 사기 리뷰·조직적 어뷰징 탐지

여러 대학의 데이터 관련 학회들과 연합체(ITDA)를 결성하고, 실제 이커머스 비즈니스 환경의 잠재 리스크를 탐지하는 것을 주제로 연합 학술제를 기획·총괄한 프로젝트입니다. 대회 주제로 채택한 "GNN 기반 사기 리뷰 탐지"가 실제로 풀 수 있는 문제인지, 가이드라인을 설계하기 전에 직접 구현해 검증해봤습니다.

## 개요

- **소속**: 서강대학교 경영데이터사이언스 학회 INSIGHT (ITDA 연합)
- **기간**: 2026.03~2026.05
- **목표**: 여러 대학의 데이터 관련 학회들과 연합체(ITDA)를 결성하고, 실제 이커머스 비즈니스 환경의 잠재 리스크를 탐지하는 것을 주제로 한 연합 학술제 개최

## 담당 역할

연합 학술제 기획 및 운영 총괄 (학회장)

## 주요 내용 및 성과

- **대회 주제 및 목표 세팅**: 대회 주제를 '그래프 신경망(GNN) 기반 사기 리뷰 및 조직적 어뷰징 탐지'로 설정하고, 단순 데이터 분석을 넘어 실제 비즈니스에 적용 가능한 MVP(데모) 제작을 대회의 최종 목표로 설정
- **가이드라인 설계 및 소통**: 모델 성능, 비즈니스 논리성, 시각적 설명력을 아우르는 명확한 평가 기준을 직접 설계하고, 참가자들의 혼선을 방지하기 위해 충분한 사전 설명과 친절한 Q&A 제공
- **대회 총괄 및 운영**: 학회장으로서 참가자 미션 안내, 예선 심사 및 본선 행사 진행 등 대회 전 과정을 총괄하며, 안정적으로 대회를 운영하고 참가자들이 서로의 분석과 인사이트를 공유하며 성장할 수 있는 환경 조성

다수의 이해관계자가 참여하는 대규모 연합 행사에서 명확한 가이드라인 세팅과 원활한 소통을 통해 혼선을 방지하고, 일정과 업무를 조율하며 조직의 목표를 성공적으로 완수하는 총괄 역량을 이 프로젝트에서 보여줬습니다.

### 요약

- 사기 리뷰 비율 13.2%인 YelpZip 데이터를 유저-리뷰-식당 간 이기종 그래프(Hetero-Graph)로 구성하고, PyTorch Geometric 기반 GNN으로 학습
- 유저(`rur`)·식당(`rpr`)·시간(`rtr`)·텍스트 유사도(`r_sim_r`) 4종의 엣지를 설계해 조직적 어뷰징 패턴을 포착
- 4-Edge 모델 PR-AUC 0.9011 달성, 이후 텍스트 유사도 기반으로 정예화하여 설명력·신뢰도를 높인 버전은 PR-AUC 0.8874
- Ablation Study 결과, 유저 연결(rur) 제거 시 성능이 71.9%p 하락 — 계정의 과거 행적이 가장 결정적인 단서로 확인
- Pyvis로 판단 근거(시간적 폭격, 복사-붙여넣기 패턴 등)를 색상별로 시각화해 "왜 사기로 판단했는지" 설명 가능한 형태로 구현

### 파이프라인

1. **데이터 전처리 및 피처 추출** — 결측치 제거, 10-Core 필터링, 리뷰 텍스트 벡터화
2. **그래프 구축** — User·Product·Time·텍스트 유사도 기준으로 리뷰 노드를 연결한 이기종 그래프 생성
3. **GNN 모델 구축 및 학습** — PyTorch Geometric으로 그래프 구조를 유지하는 마스킹 기법 학습, Weighted Loss로 불균형 보정
4. **평가 및 시각화** — PR-AUC/F1로 평가하고, 예측된 사기 리뷰 군집을 Pyvis로 시각화
